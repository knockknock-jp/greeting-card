以下の方法で画像を選択してください

  1. 画像を選択してください
  2. カメラを起動して撮影
  3. 画像をキャンパスにドラッグ&ドロップ

グリーティングカードの作り方

Step1: 画像の選択（必須）

「Picture」タブを選択してメインとなる画像を選択します。使用しているブラウザの対応に応じて、以下の方法で画像が選択できます。

画像を選択してください
 ボタンで保存されている画像を選択してください。スマートフォンの場合はカメラでその場で撮影した画像を使用する事ができます。
カメラを起動して撮影
Webカメラに対応している場合は ボタンでそのまま撮影する事ができます。キャンパス上にWebカメラの映像が表示されますので、 ボタンで撮影してください。
画像をキャンパスにドラッグ&ドロップ
PCからご利用いただいている場合は、キャンパス上に直接ファイルをドラッグ&ドロップする事で画像を選択する事もできます。

Step2: ペンツール

「Pen」タブを選択すると、ペンツールを使用する事ができます。各オプションに関しては以下の通りです。

Opacity
透過度を変更できます。100%〜25%
Thickness
ペンの太さを変更できます。
Color
ペンの色を変更できます。

Step3: スタンプツール

「Stamp」タブを選択すると、スタンプツールを使用する事ができます。各オプションに関しては以下の通りです。

スタンプの種類を変更する事ができます。
Opacity
透過度を変更できます。100%〜25%
Size
スタンプの大きさを変更できます。LL・L・M・S・SS
Flip
スタンプを反転します。Horizontal（左右反転）・Vertical（上下反転）

Step4: お名前とメッセージの入力

画像の選択とペンツールとスタンプツールを使った装飾が終わったら、 ボタンを押してください。お名前とメッセージを入力するウィンドウが表示されるので、以下の情報を入力してください。

Name
お名前を20文字以内で入力してください。
Message
メッセージを100文字以内で入力してください。
Password
パスワードを20文字以内で入力してください。このパスワードは保存したカードを削除する場合に必要になります。

Step5: カードの送信と保存

お名前とメッセージとパスワードを入力したら ボタンを押してください。サーバに作成したカードを送信、保存します。保存したカードは「Gallery」ページまたは「Slideshow」ページで共有されます。
